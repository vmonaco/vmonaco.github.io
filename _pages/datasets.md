This is a list of datasets I maintain that are available upon request.

To obtain access to any of the following datasets, send me an email with the following information:  

1.  Your name and institution
2.  The dataset(s) you would like access to
3.  Your public ssh key or Bitbucket username

Your public key or Bitbucket account will then be added to the repository. Access is provided under a [Creative Commons Attribution-NonCommercial-NoDerivatives 4.0 International License](https://creativecommons.org/licenses/by-nc-nd/4.0/). In short, the datasets are intended for research and cannot be used for commercial applications. More information is located on each dataset wiki page.

[Perez keystroke](https://bitbucket.org/vmonaco/dataset-perez-keystroke)  
Users typing with one hand and both hands to simulate an obstruction. See also the [OhKBIC data](https://bitbucket.org/vmonaco/dataset-ohkbic).  

[Four HCI tasks](https://bitbucket.org/vmonaco/dataset-four-hci-tasks)  
Mouse and keystroke data from users performing four common computing tasks.  

[Villani keystroke](https://bitbucket.org/vmonaco/dataset-villani-keystroke)  
Keystroke data from users answering open-ended questions and copying fixed text.  

[Stewart keystroke](https://bitbucket.org/vmonaco/dataset-stewart-keystroke)  
Keystroke and stylometry data from students taking online exams.  

[Bakelman keypad](https://bitbucket.org/vmonaco/dataset-bakelman-keypad)  
Keypad (on a desktop keyboard) data entry of a telephone number.  

[Coakley mobile](https://bitbucket.org/vmonaco/dataset-coakley-mobile)  
Soft keyboard and gesture input on Android mobile devices.  

[Moodle quizzes](https://bitbucket.org/vmonaco/dataset-moodle-quizzes)  
Mouse pointer motion collected during multiple choice quizzes.  

[Nursery rhymes](https://bitbucket.org/vmonaco/dataset-nursery-rhymes)  
Keystroke data from users copying several nursery rhymes.

This is a list of publicly available keystroke datasets.

[CMU Keystroke Dynamics – Benchmark Data Set](http://www.cs.cmu.edu/~keystroke/)  
51 users typing the password (.tie5Roanl)

[CMU Free vs. Transcribed Text](http://www.cs.cmu.edu/~keystroke/laser-2012/)  
20 users free and transcribed text  

[Keystroke Dynamics – Android platform](https://ms.sapientia.ro/~manyi/keystroke.html)  
42 users typing the password (.tie5Roanl) on a mobile device  

[BeiHang Keystroke Dynamics Database](http://mpl.buaa.edu.cn/detail1.htm)  
209 users typing various passwords  

[BioChaves Keystroke Databases](http://www.biochaves.com/en/download.htm)  
47 users typing four words  

[Keystroke100 Dataset](http://personal.ie.cuhk.edu.hk/~ccloy/downloads_keystroke100.html)  
100 users typing a password (try4-mbs) on a pressure-sensitive keyboard  

[GREYC Keystroke Datasets](http://www.epaymentbiometrics.ensicaen.fr/publications/resources)  
There are 3 different datasets available:  

> [133 users typing various passwords](http://www.epaymentbiometrics.ensicaen.fr/greyc-keystroke-dataset)
>
> [118 users typing various passwords and usernames](http://www.epaymentbiometrics.ensicaen.fr/greyc-web-based-keystroke-dynamics-dataset)
>
> [110 users typing five passphrases](http://www.epaymentbiometrics.ensicaen.fr/greyc-nislab-keystroke-dynamics-dataset)


[RHU KeyStroke Dynamics Benchmark Dataset](http://www.coolestech.com/rhu-keystroke/)  
51 users typing the password (rhu.university) on a mobile device  

[CITEFA Keystroke Dynamics Dataset](http://www.cse.chalmers.se/~bello/publications/) (broken link)  
63 users typing copying literature text

[ATVS-Keystroke DB](http://atvs.ii.uam.es/atvs/keystroke_db.html) (available upon request)  
63 users typing their personal information  

[LSIA Keystroke Dynamics keypress latency dataset](http://lsia.fi.uba.ar/pub/papers/kd-dataset/)  
Physicians typing patient summaries over 1 week  

[Stonybrook Keystroke Patterns as Prosody in Digital Writings](http://www3.cs.stonybrook.edu/~rbanerjee/project-pages/keystrokes/keystrokes.html)  
196 users typing free text  

[Keystroke acoustics, video, and timings](http://www.cse.msu.edu/~liuxm/typing.html) (available upon request)  
81 users typing fixed and free text  

[Detecting Age Groups using Keystroke Dynamics](http://bicakci.etu.edu.tr/dagkd/dagkd.htm)100 users typing fixed text  

[How We Type](http://userinterfaces.aalto.fi/how-we-type/)  
30 users typing various inputs with motion capture, eye tracking, and video  

[MOBIKEY Keystroke Dynamics Password Database](https://www.ms.sapientia.ro/~manyi/mobikey.html)  
54 users typing easy and strong passwords on a mobile device

[Statistical Analysis of Personality and Identity in Chats](http://dl.acm.org/citation.cfm?id=2663272) (available upon request)
50 users chatting with an unknown operator

This is a list of publicly available mouse dynamics (click and motion) datasets.

[Shen mouse behavior for static authentication](https://figshare.com/articles/Mouse_Behavior_Data_for_Static_Authentication/5619313)
56 users performing fixed operations

[Shen mouse behavior for continuous authentication](https://figshare.com/articles/Mouse_Behavior_Data_for_Continuous_Authentication/5619328)
28 users performing routine activities

[University of Victoria mouse dynamics](https://www.uvic.ca/engineering/ece/isot/datasets/#section0-3)
48 users in realistic environment over several months

[Balabit Mouse Challenge Dataset](https://github.com/balabit/Mouse-Dynamics-Challenge)
10 users motion and click behavior
