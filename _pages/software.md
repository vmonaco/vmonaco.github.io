Find me on: [github](https://github.com/vmonaco/), [bitbucket](https://bitbucket.org/vmonaco/), and [github gist](https://gist.github.com/vmonaco/).

* [frailtySurv](https://github.com/vmonaco/frailtySurv) \\
R package for simulating and fitting semiparametric shared frailty models.

* [kloak](https://github.com/vmonaco/kloak) \\
Privacy tool that obfuscates typing behavior, making keystroke biometrics less effective.

* [pohmm](https://github.com/vmonaco/pohmm) \\
Python package that implements the partially observed hidden Markov model  (POHMM)

* [Moodle BioAuth plugin](/assets/bioauth_1.0.tar.gz) \\
Moodle plugin to authenticate test takers using keystroke biometrics.

* [Moodle Competition plugin](https://bitbucket.org/vmonaco/moodle-mod_competition/)  
A Kaggle-like plugin for Moodle for data science competitions.  

* [Android BioKeyboard](https://bitbucket.org/pacebiometrics/android-biokeyboard)  
Android keyboard that logs keystrokes and other sensor input events.

* [Behavioral Biometrics Logger](https://github.com/vmonaco/biologger)  
Cross-platform (Mac, Linux, Windows) keyboard and mouse input event logger.
